# Laboratorio HTML Elements

Este es un proyecto simple de una página web usando elementos HTML personalizados. Incluye páginas como inicio, blog, contacto, galería y más.

## Archivos principales

- `index.html`: La página principal.
- `router.js`: Maneja la navegación entre páginas.
- `styles.css`: Estilos generales para la página.
- `components/`: Componentes reutilizables como header, footer y card.
- `pages/`: Las diferentes páginas del sitio.

## Páginas

- **Home**: Página de inicio.
- **Blog**: Posts.
- **Contacto**: Formulario para contactar.
- **Galería**: Imágenes y fotos.
- **Dirección**: Información de lugares del país.
